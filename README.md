# Customer Segmentation Dashboard

A modern and responsive customer segmentation dashboard built with **React**, **Vite**, and **Tailwind CSS**. The application enables users to explore customer personas through search and category-based filtering, providing an intuitive interface for visualizing customer insights.

## Features

- 🔍 Search customer personas
- 🏷️ Filter by customer categories
- 📊 Dashboard statistics
- 📱 Responsive UI
- ⚛️ Reusable React components

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript
