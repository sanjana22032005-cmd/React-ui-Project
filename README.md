# Customer Segmentation Dashboard

A modern and responsive customer segmentation dashboard built with **React**, **Vite**, and **Tailwind CSS**. The application allows users to explore customer personas through search and category-based filtering, providing an intuitive interface for visualizing customer insights and recommendations.

---

## Features

- Search customer personas using a keyword.
- Filter customers by different target audience categories.
- View detailed customer information, including demographics and preferences.
- Display personalized product recommendations for each customer segment.
- Responsive and user-friendly interface.
- Built with reusable React components for better maintainability.

---

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/YOUR_REPOSITORY_NAME.git
```

### 2. Navigate to the project directory

```bash
cd YOUR_REPOSITORY_NAME
```

### 3. Install dependencies

```bash
npm install
```

### 4. Install required packages (if setting up manually)

```bash
npm install
npm install tailwindcss @tailwindcss/vite
```

> **Note:** If you cloned this repository, running `npm install` is sufficient because all dependencies are already listed in `package.json`.

---

## Running the Application

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
